# proyectos

- [arch-hud](https://github.com/octantes/arch-hud) is a minimal, suckless based, suite of tools for arch linux
- [trackwithslate](https://github.com/octantes/trackwithslate) is a tiny and local csv based tabular data managment tool
- [shadewithseal](https://github.com/octantes/shadewithseal) is a tiny and local glsl shader playground with webgl 1.0 support
